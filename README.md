# Sales Data Analysis Dashboard

## Project Overview
This project analyzes sales data to identify trends, top-performing regions, and product categories. The dashboard helps in understanding business growth and profit margins.

## Dashboard Preview
![Dashboard Preview](Dashboard%20Screenshot.png)

## Key Insights
- **Total Sales:** 429K
- **Top Region:** Chennai Leads in Sales
- **Top Category:** Electronics Accounts for 71% of total sales
- **Sales Trend:** Sales peaked towards the end of the month (Jan 20-25)

## Tools Used
- **Power BI** (Data Visualization)
- **Power Query** (Data Cleaning)
- **Excel** (Data Source)
